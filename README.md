# 2本腕ベルヌーイバンディット問題における Random, Greedy, ε-Greedy 方策の比較

このリポジトリは、2本腕ベルヌーイバンディット問題において Random, Greedy, ε-Greedy の3手法を比較した小規模実験のコードと論文草稿をまとめたものです。

## 概要

本プロジェクトの目的は、研究を行う練習として「技術ノートを書き、実験を行い、結果を論文としてまとめる」一連の流れを経験することでした。

## 問題設定
- 2本腕ベルヌーイバンディット
- 腕1の成功確率: 0.7
- 腕2の成功確率: 0.5
- 成功報酬: 1
- 失敗報酬: 0
- 各手法 100 trial
- 1 trial あたり 1000 step
- 比較手法:
  - Random
  - Greedy
  - ε-Greedy

## 主な結果

本設定では ε-Greedy が最も高い平均累積報酬を示し、累積後悔も最も小さかった。
